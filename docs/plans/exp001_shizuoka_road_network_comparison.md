# 実験計画: 静岡県3都市の道路ネットワーク比較分析

## 実験ID

exp001

## 目的

静岡県3都市（焼津市・静岡市・浜松市）の道路ネットワーク特性を、OSMnxで取得した道路データと8指標を用いて定量的に比較する。防災（津波避難・地震避難）と観光（回遊性・歩きやすさ）の2つの観点から考察し、各都市の道路構造の特徴を明らかにする。

## 背景

焼津市訪問に際し、焼津市と周辺都市の道路ネットワークの違いを定量的に把握したい。焼津市は駿河湾に面した漁港都市であり、津波リスクが高い。一方、静岡市は県庁所在地、浜松市は工業都市と、それぞれ異なる都市特性を持つ。これら3都市の道路構造の違いを、防災と観光の2観点から分析する。

## 理論的枠組み

### 劉（2016）の道路ネットワーク解析手法

劉澤の博士論文「道路ネットワーク解析に基づく日本都市の歴史地区の街路空間構成に関する研究」（九州大学, 2016）では、日本の16歴史的地区の道路ネットワークをグラフ理論に基づき、以下の3つの特性で定量的に分析している:

1. **回遊性**: 循環路の多さを評価する指標群（μ, α, β, γ指標）
2. **アクセス性**: 最短距離と密度に基づく到達性（Di, Dl道路密度, Dc交差点密度）
3. **迂回性**: 道路距離と直線距離の比率（迂回率A）

### 劉の4類型

- **類型I**（島嶼・山間部集落）: 主要道路の回遊性が低く、細街路が回遊性向上に大きく寄与
- **類型II**（沿岸・島嶼近都市部）: 類型Iに類似するが、道路網が比較的粗
- **類型III**（都市部地区）: 格子型道路網で回遊性・アクセス性が高い。細街路は袋路状で回遊性に寄与しない
- **類型IV**（寺内町）: 碁盤目状町割で細街路が南北を貫通し、回遊性に大きく貢献

### 本実験への適用

劉の3観点を本研究の8指標に対応させ、分析を構造化する:

| 劉の観点 | 対応する8指標 | 分析上の意味 |
|---------|------------|------------|
| **回遊性** | alpha_index, degree_centrality_mean, basic_streets_per_node_avg, basic_self_loop_proportion | 道路網の循環路の豊富さ・分岐の多様性 |
| **アクセス性** | closeness_centrality_mean, basic_clean_intersection_density_km | ネットワーク全体の到達性・交差点の密度 |
| **迂回性** | avg_circuity_A, basic_circuity_avg | 道路距離と直線距離の乖離度 |

## 使用指標（8指標）

### 回遊性グループ

| 指標 | 定義 | 防災上の意味 | 観光上の意味 |
|------|------|------------|------------|
| alpha_index | (e - n + p) / (2n - 5) | 経路の冗長性。高いほど代替避難路が多い | 循環ルートの多さ |
| degree_centrality_mean | 次数中心性の平均 | 交差点の接続性 | 方向選択の豊富さ |
| basic_streets_per_node_avg | ノードあたり平均街路数 | 分岐の多さ | 街路の多様性 |
| basic_self_loop_proportion | 自己ループ辺の比率 | ロータリー等の計画的道路 | 周回型ルートの存在 |

### アクセス性グループ

| 指標 | 定義 | 防災上の意味 | 観光上の意味 |
|------|------|------------|------------|
| closeness_centrality_mean | 近接中心性の平均 | 避難場所への到達性 | 歩きやすさ |
| basic_clean_intersection_density_km | km²あたり交差点密度 | 避難ルート分岐点の多さ | ブロックの細かさ |

### 迂回性グループ

| 指標 | 定義 | 防災上の意味 | 観光上の意味 |
|------|------|------------|------------|
| avg_circuity_A | dijkstra距離/ユークリッド距離の平均 | 迂回度。1に近いほど迅速な避難 | 経路の直進性 |
| basic_circuity_avg | OSMnx算出の平均迂回率 | 同上 | 同上 |

## 分析設計

### 1. 市全域分析

各市の行政区域全体の道路ネットワーク（`network_type='drive'`）を取得し、8指標を算出。

対象都市:
- 焼津市（`"焼津市, 静岡県, Japan"`）
- 静岡市（`"静岡市, 静岡県, Japan"`）
- 浜松市（`"浜松市, 静岡県, Japan"`）

### 2. サブエリア分析（防災）

各市を海岸線からの距離で「沿岸部（2km以内）」「内陸部（2km超）」に分割し、沿岸部の避難路ネットワーク特性を比較。

- 海岸線データ: `ox.features_from_place()` の `natural=coastline` で取得
- 各ノードの海岸線からの距離を計算し、2km以内を沿岸部として分類
- 沿岸部・内陸部のサブグラフを抽出して指標算出

### 3. サブエリア分析（観光）

各市の主要駅周辺800mの歩行者ネットワーク特性を比較。

対象駅:
- 焼津駅（34.8671, 138.3225）
- 静岡駅（34.9717, 138.3891）
- 浜松駅（34.7038, 137.7350）

`network_type='walk'` と `network_type='drive'` の両方で取得し、細街路の効果を分析。

### 4. 細街路効果分析

drive vs walk での指標差を比較し、劉の類型との対照を行う:
- 都市部（静岡駅・浜松駅）では細街路が袋路状で回遊性に寄与しにくいか
- 漁港町（焼津駅）では細街路が回遊性を向上させるか

## 仮説

1. **沿岸部の道路ネットワークは内陸部と比べて回遊性が低く迂回性が高い**（劉の沿岸部集落の知見に基づく）
2. **都市規模が異なる3市の駅前地区の回遊性に差異がある**（劉の類型III都市部と漁港町の差異に着想）
3. **細街路の回遊性への寄与度は都市の性格により異なる**（劉の細街路効果の知見に基づく）
4. **8指標による多面的評価により都市固有のネットワーク特性を識別できる**
